# Python 快速入门

Python 是一种广泛使用的高级编程语言，以其简洁易读的语法而闻名。

## 安装 Python

### macOS

```bash
brew install python3
```

### 验证安装

```bash
python3 --version
pip3 --version
```

## 基础语法

### 变量和数据类型

```python
# 字符串
name = "Python"

# 整数
age = 30

# 浮点数
price = 9.99

# 布尔值
is_active = True
```

### 列表和字典

```python
# 列表
fruits = ["apple", "banana", "orange"]

# 字典
person = {
    "name": "Alice",
    "age": 25
}
```

## 常用命令

```bash
# 运行 Python 脚本
python3 script.py

# 安装包
pip3 install package_name

# 查看已安装的包
pip3 list
```
